### 外卖
好！中午等外卖迟迟等不到信息，以为还没送到，结果已经到了，还好我眼尖，不然就白等了......
总结出校内配送的几大特点：

### 当是只道
十年前，我抗拒早睡，只想多看一眼电视，不理解大人为何总是困倦。

八年前，我觉得上学是种束缚，只想永远留在妈妈身边。

五年前，我相信身边的朋友会是一辈子。

两年前，我以为和某个人的纠缠永远不会结束。

一年前，我说人活着开心就好，可后来那些说好永远在一起的人，聊天记录只剩空白；

约定走很久的朋友，终究成了陌生人。

原来“早悟兰因，不结絮果”。

改变总是静默发生——直到某天，才发现自己不再是当初的自己。

如今我什么也不再奢求，只渴望一夜安稳的睡眠。

终于懂得：这世上没有什么是永恒的，除了“一切都在变”本身。

我们都在流转的时光里，悄悄告别了一个又一个从前的自己。
